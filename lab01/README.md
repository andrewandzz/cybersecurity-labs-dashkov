# Лабораторна робота № 1

Для того, щоб запустити програму на виконання, достатньо відкрити файл **index.html** в будь-якому браузері.

## Приклади роботи програми

Початковий вигляд програми має бути таким:

![Початковий вигляд програми](screenshots/img01.png)

Далі потрібно ввести відповідні дані в усі необхідні поля та натиснути кнопку "Проаналізувати пароль". Після цього програма видасть результат аналізу пароля з оцінкою за десятибальною шкалою, деталями аналізу та рекомендаціями щодо покращення надійності пароля.

Наводимо приклад введених даних:

![Приклад введення даних](screenshots/img02.png)

Ми бачимо результати аналізу пароля для введених даних:

![Аналіз слабкого пароля](screenshots/img03.png)

А такі результати аналізу ми отримуємо для пароля надійності нижче середнього:

![Аналіз середнього пароля](screenshots/img04.png)

Це - результати аналізу надійного пароля:

![Аналіз надійного пароля](screenshots/img05.png)
