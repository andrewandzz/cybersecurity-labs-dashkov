# Лабораторні роботи з курсу &laquo;Безпека програм та даних&raquo;

Цей репозиторій містить вихідний код програм до лабораторних робіт з навчальної дисципліни &laquo;Безпека програм та даних&raquo;. Крім файлів вихідного коду, кожна папка також містить інструкції з запуску кожної програми та скріншоти їх роботи.

## Структура репозиторію

Репозиторій структурований згідно з вимогами - кожна лабораторна робота знаходиться в окремій папці.

| Папка   | Назва лабораторної роботи             |
| :------ | :------------------------------------ |
| `lab01` | Аудит власної цифрової безпеки        |
| `lab02` | Захист особистих повідомлень          |
| `lab03` | Невидимі дані у власних файлах        |
| `lab04` | Особистий цифровий підпис             |
| `lab05` | Захищена електронна пошта             |
| `lab06` | Етичний хакінг власного застосунку    |
| `lab07` | Комплексний захист особистого проекту |

## Інструкції з використання

Всі створені програми є HTML сторінками, які не потребують запуску сервера або будь-яких інших додаткових налаштувань, тому можуть бути запущені одразу в будь-якому сучасному браузері шляхом відкриття файлу `index.html` відповідної папки.
